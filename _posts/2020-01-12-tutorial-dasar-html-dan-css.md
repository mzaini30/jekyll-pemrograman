Buat file `index.html` lalu isi dengan teks berikut ini:

```html
<p>Hello World</p>
```

Apa yang terjadi?

Yap, sebuah tulisan Hello World.

Kemudian coba ubah menjadi:

```html
<style>
  p {
    color: green;
  }
</style>
<p>Hello World</p>
```

Sekarang, Hello World menjadi berwarna hijau.
